## fmt

打印函数：详看[go语言标准库](https://studygolang.com/pkgdoc)

1. ```go
   Print：直接输出
   PrintIn：输出后加换行
   Printf：格式化输出
   ```

2. ```go
   Scan：扫描标准输入的文本，由空白符分隔的值传递到本函数的参数中
   Scanf：指定占位符格式获取
   ScanIn：换行
   ```

3. ```go
   %d          十进制整数
   %x, %o, %b  十六进制，八进制，二进制整数。
   %f, %g, %e  浮点数： 3.141593 3.141592653589793 3.141593e+00
   %t          布尔：true或false
   %c          字符（rune） (Unicode码点)
   %s          字符串
   %q          带双引号的字符串"abc"或带单引号的字符'c'
   %v          变量的自然形式（natural format）
   %T          变量的类型
   %%          字面上的百分号标志（无操作数）
   ```

# flag

一系列解析命令行参数的功能接口。

## String()

````go
flag.String(name string, value string, usage string) *string
````

含义：优先从命令行参数中获取`name`的值，没有即为`string`，`usage`是注释。返回值为字符串的指针。

## Parse()

```go
flag.Parse()
```

含义：解析命令行参数到定义的flag。

# path/filepath

## Abs()

```go
filepath.Abs(path string) (string, error)
```

获取绝对路径。

## Dir()

```go
filepath.Dir(path string) string
```

将路径去掉最后一个元素，返回字符串。

## Base()

```go
filepath.Base(path string) string
```

返回指定路径的最后一个元素。

# Content

## WithValue()

```go
content.WithValue(parent Context, key, val interface{}) Context
```

给context 设置上一个值。

会产生一个新的ctx，值存储在这个新的ctx上。

## WithCancel()

```go
content.WithCancel(parent Context) (ctx Context, cancel CancelFunc)
```

创建一个可取消的context。

## WithTimeout()

```go
content.WithTimeout(parent Context, timeout time.Duration) (Context, CancelFunc)
```

限定下游超时时间

## Deadline()

```go
content.Deadline() (deadline time.Time, ok bool)
```

获取运行截止时间

## Done()

```go
content.Done() <-chan struct{}
```

用于判断是否已经终止/取消

## Err()

```go
content.Err() error
```

 若已经终止/取消 将返回error

## Value()

```go
content.Value(key interface{}) interface{}
```

查找key的值
