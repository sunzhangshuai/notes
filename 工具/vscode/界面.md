# 界面

## 欢迎使用

### 启动

```shell
# 新建文件
command + N
# 打开文件夹
command + O
```

### 学习

- 查找并运行所有命令

- 界面概述

  - 查找并运行所有命令

    ```shell
    shift + command + P
    ```

  - 界面概览

    - 左侧

      ```shell
      # 文件资源管理器
      shift + command + E
      # 跨文件搜索
      shift + command + F
      # 源代码管理
      control + shift + G
      # 启动与调试
      shift + command + D
      # 管理扩展
      shift + command + X
      ```

    - 上侧

      ```shell
      # 查找并运行所有命令
      shift + command + P
      ```

    - 左下侧

      ```shell
      # 查看错误和警告
      shift + command + M
      ```

    - 下侧

      ```shell
      # 查看错误和警告
      shift + command + M
      ```

# 显示所有命令

F5
# 切换终端
control + `
