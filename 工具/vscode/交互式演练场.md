# 交互式演练场

## 多光标编辑

### 框选

1. 放置光标
2. *Option + Shift*
3. 拖动光标选中矩形

### 增加一个光标

*Command + Option + 上下键*

### 选中所有和当前选中相同的字符串

*Command + Shift + L*

## 自动补全

*Control + Space*

**容易和切换输入法冲突**

## 行操作

快速移动行

### 复制光标所在行

*Shift + Option + 上*		向上复制

*Shift + Option + 下*		向下复制

### 移动光标所在行

*Option + 上*		向上移动

*Option + 下*		向下移动

### 删除光标所在行

*Shift + Command + K*
