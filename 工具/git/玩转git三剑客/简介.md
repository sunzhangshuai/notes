# git

[官网地址](https://git-scm.com/)

## 特点

1. 最优的存储能力
2. 非凡的性能
3. 开源
4. 易备份
5. 支持离线操作
6. 易定制工作流程

## 命令

1. 查看git版本

   ```shell
   git --version
   ```

2. 配置

   ```shell
   # 最小配置
   git config [--local|--global|--system] user.name your_name
   git config [--local|--global|--system] user.email XXX@XXX.com
   
   # 查看配置
   git config --list [--local|--global|--system]
   ```

   1. --local[**默认**]：对当前用户某个仓库有效，**并且只能在某仓库下使用**。
   2. --system：对系统所有用户有效。
   3. --global ：对当前用户所有仓库有效。

3. 创建git仓库

   ```text
   git init [projectName]
   ```

   1. 有项目名：先创建*projectName*文件夹，在此文件夹下创建`.git`。
   2. 缺省项目名：在当前目录下创建`.git`。 

把工作目录下的文件提交到暂存区

```shell
git add [files...|-u]
```

1. files : 把指定的一个或多个文件提交到暂存区。
2. -u: 只把之前暂存过的文件再提交到暂存区。

告知暂存区 工作目录下已经删除的文件

```shell
git rm file... 
```

把暂存区的文件提交到本地仓库

```shell
git commit -m "msg" 
```

删除文件，放到工作目录

```shell
git rm file... --cached
```

直接删除文件

```shell
git rm file... -f
```

查看提交版本历史

```shell
git log
```
