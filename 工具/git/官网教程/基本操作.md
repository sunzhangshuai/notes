# 移除文件

1. ```shell
   git rm files 
   ```

   1. 如果在工作目录中删除了文件，需要执行该命令从跟踪清单中删除。**最终生效还需 commit**。

   2. 效果：将文件从跟踪清单中删除，并且在工作目录下删除。**最终生效还需 commit。**

   3. **注意**：如果该文件在工作目录或者暂存区中做过修改，需要使用 -f 强制删除。

      ```shell
      git rm files -f
      ```

   4. files可以使用正则。

      ```shell
      git rm log/\*.log
      git rm \*idea
      ```

2. ```shell
   git rm files --cached
   ```

   效果：将文件从跟踪清单中删除，但在工作目录中保留。
